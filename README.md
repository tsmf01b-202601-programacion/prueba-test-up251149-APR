# Examen Segunda Oportunidad Python – Unidad 1 (Días 1 al 4)

Este examen evalúa los conocimientos adquiridos en los días 1 al 4 del curso **30 Days of Python**.  
Resuelve los ejercicios directamente en el archivo `examen.py`.

---

## 📝 Instrucciones

- 📥 El archivo `examen.py` ya contiene los datos de entrada para cada ejercicio.
- 🧠 Cada ejercicio se selecciona escribiendo el número correspondiente (del 1 al 10) cuando el programa inicia.
- ✍️ Escribe tu código **solo donde se indica con el comentario** `# Tu código aquí`.
- 🔄 Usa únicamente lo que se ha visto en clase (sin ciclos ni estructuras avanzadas).
- ✅ Debes usar `print()` para mostrar el resultado de cada ejercicio.

---

## 🚫 Está prohibido

- ❌ Usar GitHub Copilot u otras herramientas de autocompletado de código inteligente.
- ❌ Navegar el repositorio otros repositorios.
- ❌ Abrir este repositorio desde cualquier entorno que **no sea Codespaces en Visual Studio Code**.
- ❌ Pedir ayuda externa o compartir tu código con otros compañeros.
- ❌ Usar `if`, `for`, `while`, `def`, u otras estructuras que aún no se han visto.
